# Quora Question Pairs Project

This project is based on the Quora Question Pairs Kaggle competition, where the goal is to determine whether two questions are duplicates of each other.

## Project Overview

This repository contains notebooks for exploring, preprocessing, and modeling duplicate question detection using natural language processing techniques.

### Notebooks
- duplicate-question-pairs.ipynb
- suplicate-question-pairs-v2.ipynb

## What I Did

- Loaded and explored the Quora Question Pairs dataset
- Performed data cleaning and preprocessing
- Applied feature engineering techniques such as:
  - question length
  - number of words
  - common word count
  - word share ratio
  - text-based vector features
- Built and tested machine learning models for duplicate detection
